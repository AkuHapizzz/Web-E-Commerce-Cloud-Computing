# Web E-Commerce Cloud Computing

This repository contains the source code for the ITB Speedshop e-commerce platform, developed as a project for the Cloud Computing course.

## Project Structure

The project is organized as follows:

- `itb-speedshop/`: The main Laravel application directory containing the backend and frontend source code.
